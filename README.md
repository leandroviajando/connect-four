# [connect-four](https://www.coursera.org/projects/connect-four-game-in-python-using-pygame)

## Installation

```bash
git lfs install
git lfs track "*.png"

python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

## Play Game

```bash
source .venv/bin/activate
python main.py
```
